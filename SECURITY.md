## Reporting a Vulnerability

If you discover a security vulnerability in Zyra, please report it responsibly.

- Do **not** open a public issue for security vulnerabilities
- Instead, contact the project maintainers directly

Details to include if possible:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

Reports will be reviewed and addressed as soon as possible.

## Disclosure Policy

Security issues will be handled privately until a fix is available.  
Once resolved, a security advisory may be published.
